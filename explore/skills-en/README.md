# English Skills Guide

Each skill in this folder is independently usable and does not require `skills-zh`.

## Quick Start

1. Enter any skill folder.
2. Open `SKILL.md` and run the quick-start command.
3. Use local `scripts/`, `prompts/`, and `examples/` in the same skill folder.

## Folder Conventions

- `SKILL.md`: practical usage guide
- `prompts/`: prompt text for the skill
- `scripts/`: parsers, generators, runners
- `examples/`: sample input/output files
- `output-templates/`: output templates when provided
- `references/`: scope and framework notes when provided

## Independence Rule

- English skills are runnable from this folder only.
- No cross-folder dependency is required.
