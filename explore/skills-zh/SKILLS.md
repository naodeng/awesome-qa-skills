# 中文 Skills

这是中文技能总目录。所有技能均可在本目录内独立运行。

## 已提供技能

- requirements-analysis-plus：需求分析增强版（多格式解析 + 分析结论）
- testcase-writer-plus：测试用例编写增强版（多格式输入 + 多格式输出）
- test-strategy-plus：测试策略增强版（多文档输入 + 多格式输出）
- test-case-reviewer-plus：测试用例评审增强版（多文档输入 + 多格式输出）
- performance-test-k6：k6 性能测试方案与脚本
- performance-test-gatling：Gatling 性能测试方案与脚本
- api-test-supertest：Supertest 接口自动化（多格式接口输入）
- api-test-bruno：Bruno 接口自动化（多格式接口输入）
- api-test-pytest：Pytest 接口自动化（多格式接口输入）
- api-test-restassure：Rest Assured 接口自动化（多格式接口输入）

## 使用原则

- 只看中文目录即可完成使用。
- 每个技能都包含自己的脚本、提示词、示例和模板。
- 默认输出优先使用 Markdown；按需切换 Word/JSON/Excel（TSV）。
