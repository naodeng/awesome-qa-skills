# 中文 Skills 使用指南

本目录下的每个技能都可以独立使用，不依赖 `skills-en`。

## 你可以直接这样用

1. 进入任意技能目录。
2. 按 `SKILL.md` 的“快速开始”执行示例命令。
3. 输入文件、脚本、模板、示例都在当前技能目录内。

## 目录说明

- `SKILL.md`: 技能说明、输入输出、执行步骤
- `prompts/`: 中文提示词
- `scripts/`: 解析、生成、执行脚本
- `examples/`: 可直接跑的示例输入/输出
- `output-templates/`: 多格式输出模板（按技能提供）
- `references/`: 规范和落地建议（按技能提供）

## 独立性约定

- 中文技能以当前目录文件为准，不需要跳转英文目录。
- 如需中英文同步，使用“功能对齐”方式，不做路径依赖。
