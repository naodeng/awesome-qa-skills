# performance-test-k6（中文）

## 适用场景

适合活动上线前验证吞吐、响应和失败率，快速发现容量瓶颈。

## 技能目标

提供 k6 性能测试范围、脚本样例和执行入口，覆盖负载/压力/尖峰/稳定性。

## 输入

- 可选：目标地址、鉴权信息、并发参数、环境变量文件

## 输出

- 输出：性能执行结果与汇总数据

## 前置准备

- 安装 k6
- 配置可访问的目标环境
- 先从 smoke 或 load 类型开始

## 快速开始

```bash
bash scripts/run-tests.sh load
```

## 离线演示（本地可跑）

```bash
bash scripts/run-local-smoke.sh
```

## 结果判断

- 命令可启动并进入执行流程
- 能看到结果摘要和阈值判定

## 常见问题

- 若失败率很高，先检查目标地址和网络连通
- 若命令不存在，先安装 k6

## 目录说明

- `prompts/`：中文提示词
- `scripts/`：执行脚本
- `examples/`：示例输入输出
- `output-templates/`：输出模板（如目录存在）
- `references/`：规范说明（如目录存在）

## 独立使用声明

- 本技能可脱离英文目录单独使用。
- 运行所需文件全部位于当前技能目录内。
- 文档、提示词、示例均为中文可读版本。
