# skills-en

English skill partition directory.

- `testing-types/`: English testing-type skills (display names without `-en`)
- `testing-workflows/`: English testing-workflow skills (display names without `-en`)

Examples:

- `testing-types/functional-testing` -> `../../testing-types/functional-testing-en`
- `testing-workflows/daily-testing-workflow` -> `../../testing-workflows/daily-testing-workflow-en`

Note: this directory is a symlink-based language view mapped to `skills/testing-types` and `skills/testing-workflows`, mainly for navigation.
For editing/install scripts/release packaging, use core directories under `skills/testing-types` and `skills/testing-workflows`.
