# 输出格式说明（Workflow）

默认输出：Markdown。

如需其他格式，请在需求末尾追加：

- Excel（制表符分隔）：请以 Excel 可粘贴的制表符分隔表格输出。
- CSV：请以 CSV 格式输出。
- JSON：请以 JSON 形式输出。

建议：

- 路由结论给人看：Markdown 或 Excel。
- 路由结论给系统用：JSON。

---

# Output Format Guide (Workflow)

Default output: Markdown.

If you need another format, append one request sentence at the end of your input:

- Excel (tab-separated): Please output as tab-separated table for pasting into Excel.
- CSV: Please output as CSV.
- JSON: Please output as JSON.

Recommendations:

- Human-readable routing result: Markdown or Excel.
- System-automation routing result: JSON.
