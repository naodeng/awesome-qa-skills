# 日常测试工作流 — 提示词对照

执行各步骤时使用**本目录 `prompts/`** 下对应文件。下表为步骤与提示词文件对照。

| 步骤 | 提示词文件 | 用途 |
|------|------------|------|
| 早晨审查测试计划 | requirements-analysis.md, test-strategy.md | 审查当日故事与高风险区域 |
| 设置测试环境 | automation-testing.md, test-strategy.md | 流水线与测试数据 |
| 测试用例创建 | test-case-writing.md, requirements-analysis.md, functional-testing.md | 新功能/缺陷修复用例 |
| 测试自动化 | automation-testing.md, api-testing.md, ai-assisted-testing.md | 编写与维护自动化 |
| 探索性测试 | manual-testing.md | 章程与探索会话 |
| 缺陷上报 | bug-reporting.md | 缺陷报告 |
| 可视化 / E2E | accessibility-testing.md, functional-testing.md | 视觉与端到端 |
| 下午审查 | test-reporting.md, test-strategy.md | 覆盖率与质量指标 |
