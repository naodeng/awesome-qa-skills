## Excel 公式示例

### 自动计算完成度
```excel
=IF(D2=0, 0%, E2/D2)
```

### 自动计算风险等级
```excel
=IF(AND(C2="高", D2="高"), "高", IF(OR(C2="高", D2="高"), "中", "低"))
```

### 自动统计缺陷数量
```excel
=COUNTIF(缺陷统计!H:H, "P0")
```

### 自动计算达成率
```excel
=IF(B2=0, "N/A", TEXT(C2/B2, "0%"))
```

## 条件格式设置

### 状态颜色
- 完成: 绿色背景
- 进行中: 黄色背景
- 计划中: 灰色背景
- 延期: 红色背景

### 优先级颜色
- P0: 深红色
- P1: 橙色
- P2: 黄色
- P3: 绿色
- P4: 灰色

### 进度条
使用数据条显示完成度百分比

## 数据透视表

### 按模块统计测试用例
- 行: 模块
- 列: 测试类型
- 值: 计数

### 按人员统计工作量
- 行: 姓名
- 列: 任务
- 值: 工作量求和

### 按日期统计缺陷趋势
- 行: 日期
- 列: 缺陷状态
- 值: 计数

## 图表建议

1. **甘特图**: 测试进度时间线
2. **饼图**: 测试类型分布
3. **柱状图**: 每日缺陷趋势
4. **折线图**: 缺陷累积曲线
5. **雷达图**: 测试度量达成情况

## 使用技巧

1. **冻结首行**: 便于查看表头
2. **数据验证**: 限制输入值（如状态、优先级）
3. **保护工作表**: 防止误修改公式
4. **自动筛选**: 快速查找数据
5. **条件格式**: 突出显示关键信息
6. **命名区域**: 简化公式引用
7. **宏录制**: 自动化重复操作

- **PDF**: 用于正式报告
- **CSV**: 用于数据交换
- **在线共享**: Google Sheets 实时协作
