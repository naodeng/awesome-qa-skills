## 使用指南

### 1. 生成报告
```javascript
// Node.js 示例
const fs = require('fs');
const template = fs.readFileSync('test-report-template.html', 'utf8');

// 替换模板变量
const report = template
  .replace('{{totalCases}}', testResults.total)
  .replace('{{passedCases}}', testResults.passed)
  .replace('{{failedCases}}', testResults.failed);

fs.writeFileSync('test-report.html', report);
```

### 2. 集成到 CI/CD
```yaml
# .github/workflows/test.yml
- name: Generate HTML Report
  run: |
    npm run test:report
    
- name: Upload Report
  uses: actions/upload-artifact@v2
  with:
    name: test-report
    path: test-report.html
```

### 3. 添加图表
```html
<!-- 使用 Chart.js -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
<canvas id="testChart"></canvas>
<script>
new Chart(document.getElementById('testChart'), {
  type: 'pie',
  data: {
    labels: ['通过', '失败', '跳过'],
    datasets: [{
      data: [22, 2, 1],
      backgroundColor: ['#28a745', '#dc3545', '#ffc107']
    }]
  }
});
</script>
```

## 最佳实践

1. **响应式设计**: 支持移动设备查看
2. **打印友好**: 使用 @media print 优化打印样式
3. **交互性**: 可折叠的测试用例详情
4. **可视化**: 使用图表展示统计数据
5. **链接**: 关联缺陷跟踪系统
6. **截图**: 包含失败用例的截图
7. **历史对比**: 显示趋势图
8. **导出功能**: 支持导出为 PDF
