## 使用指南

### 选择合适的格式

| 格式 | 适用场景 | 优势 | 劣势 |
|------|----------|------|------|
| HTML | 人工查看、演示 | 可视化好、交互性强 | 不易解析 |
| Markdown | 文档、Git | 易读、版本控制友好 | 格式有限 |
| JSON | API集成、程序处理 | 结构化、易解析 | 不易人工阅读 |
| XML | 企业工具集成 | 标准化、工具支持好 | 冗长 |
| CSV | 数据分析、Excel | 简单、通用 | 结构扁平 |
| YAML | CI/CD、配置 | 简洁、易读 | 解析器差异 |

### 格式转换

```bash
# JSON 转 HTML
node scripts/json-to-html.js report.json > report.html

# XML 转 JSON
xq . report.xml > report.json

# CSV 转 JSON
csvtojson report.csv > report.json

# YAML 转 JSON
yq eval -o=json report.yaml > report.json
```

### CI/CD 集成

```yaml
# .github/workflows/test.yml
- name: Run Tests
  run: npm test

- name: Generate Reports
  run: |
    npm run test:report:html
    npm run test:report:json
    npm run test:report:junit

- name: Upload Reports
  uses: actions/upload-artifact@v2
  with:
    name: test-reports
    path: |
      reports/*.html
      reports/*.json
      reports/*.xml
```

## 最佳实践
