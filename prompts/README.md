# 工作流所用提示词

本目录保存仓库根级别的公共提示词清单，现已按语言拆分：

- 中文：`prompts/zh/`
- 英文：`prompts/en/`

## 文件规则

- 中文提示词直接放在 `prompts/zh/`
- 英文提示词直接放在 `prompts/en/`
- 英文文件名不再使用 `_EN` 后缀

## 当前覆盖

- accessibility-testing
- ai-assisted-testing
- api-testing
- automation-testing
- bug-reporting
- functional-testing
- manual-testing
- mobile-testing
- performance-testing
- requirements-analysis
- security-testing
- test-case-reviewer
- test-case-writing
- test-reporting
- test-strategy

## 使用方式

- 中文场景：打开 `prompts/zh/` 下对应文件
- 英文场景：打开 `prompts/en/` 下对应文件
- 具体技能的正式入口仍以各 skill 目录内的 `prompts/` 为准
